This project was built to demonstrate an amazing function of the 'yes' command in the linux terminal...
The 'yes' command repeats any text you type forever until you stop it.
How it works:
> When you type
yes hello
It will keep printing:
hello
hello
hello
hello
hello
...
Consider this visualization that the 'yes' command is built in, Now type in anything and play around with the button!
